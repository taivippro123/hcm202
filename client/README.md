# SlideBook

Slide 3d Book for HCM

## Features

- 3D Interactive Book with React Three Fiber
- Mobile Responsive Design
- Audio Page Flip Effects
- Multiple Background Options
- Touch-friendly Navigation

## Tech Stack

- React 18
- React Three Fiber
- Three.js
- Tailwind CSS
- Vite

## Installation

```bash
npm install
npm run dev
```

## Deploy

Deploy to Vercel for production use.
